1、在Window安装GIT很简单，到官网中下载安装包，一路next就行。
2、安装完成后设置Git提交代码的用户及邮箱，命令如下：
	git config --global user.name "你的用户名"
	git config --global user.email "你的邮箱"
	需要注意：
		git config命令中的--global参数，用了这参数，表示当前这台机上所有的仓库都会使用这个用户名及邮箱来表示身份。也可以单独指定用户及邮箱
上面相当于准备工作，接下来继续学习。

1、创建一个空目录：git_test，在目录中右键选择Git Bash进入Git命令行，如果需要看到当前所在目录，使用pwd命令则可以看到当前所有的目录。
2、将该目录变成Git可以管理的仓库，使用git init命令，命令执行后，可以看到一个.git的隐藏目录
3、接下来进入怎样将文件放入仓库中，创建一个readme.txt文件。
	将创建的文件添加到仓库中：git add readme.txt
4、将放入仓库的文件提交到仓库中，命令如下：
	git commit -m "wrote a readme file"
	命令中的-m参数后面输入的表示这次提交文件的说明，可以是任意内容，如果以后想找历史版本，可以根据提交文件的描述找更为快捷。
5、如果需要看到当前仓库的状态，则可以如下命令：
	git status
6、如果想看仓库的文件与现在修改的文件有什么不同，则使用该命令查看：
	git diff
7、在之前已经提交了几次文件了，这个时候我们可能想看看之前修改了什么东西，那该怎么办呢？git中还有一个命令可以查到历史纪录，命令如下：
	git log
	输入命令后，则可以看到详细的修改信息，如果你不需要看到这么多的内容，可以在后面加上如下参数：
	git log	--pretty=oneline
	就会输出版本号及修改文件提交时的修改说明了。
8、当我们将文件修改了很多次，想回退到历史版本，这个时候我们可以使用版本回退命令：
	git reset --hard HEAD^
	这里的HEAD表示当前版本，^表示上一个版本，^^表示上上个版本，那么如果上100个版本，那么该怎么表示呢？使用HEAD~100即可.
